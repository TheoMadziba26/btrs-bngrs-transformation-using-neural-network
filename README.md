**BTRS → BNGRS Coordinate Transformation Suite**

A production-ready web application for converting coordinates between Botswana's Traditional Reference System (BTRS) and National Geodetic Reference System (BNGRS) using ensemble machine learning.

**Key Features:**
-  Sub-meter accuracy (RMSE < 0.5m)
-  Real-time single point & batch processing  
-  Comprehensive residual analysis & statistics
-  Interactive Leaflet map with vector visualization
-  Professional reporting (CSV, JSON, GeoJSON)
-  Ensemble: XGBoost + Neural Network averaging

**Tech Stack:**
- Backend: Flask, TensorFlow, XGBoost, scikit-learn
- Frontend: HTML5, Leaflet.js, Vanilla JS
- ML: Ensemble regression with feature engineering

**Use Cases:**
- Land surveying & cadastral mapping
- GIS data migration & integration  
- Geodetic research & validation
- Educational demonstrations
